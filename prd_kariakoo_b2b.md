# PRD: Kariakoo Soko B2B - Alibaba for Tanzania
**Version:** 1.0 - Concept PRD
**Date:** 2026-06-23
**Market:** Dar es Salaam / Tanzania / East Africa
**Platforms:** iOS & Android

## 1. Product Vision
Kariakoo Soko B2B is the Alibaba of Tanzania. A verified wholesale marketplace connecting Kariakoo importers & local manufacturers in Dar es Salaam with duka owners, retailers, and resellers across Tanzania.

We solve 3 core trust problems in Tanzanian wholesale: Fake suppliers, COD fraud, and upcountry logistics chaos.

## 2. Personas
1.  **Fatuma - Buyer Retailer:** Duka owner in Mwanza. Buys 50-200pcs vitenge, shoes monthly. Pain: Traveling to Kariakoo is 15hr bus, expensive, risky.
2.  **Ahmed - Supplier:** Kariakoo Gold Supplier. Electronics importer at Kariakoo Msimbazi St. Has BRELA + TIN. Wants upcountry buyers without agents.
3.  **Juma - Logistics Agent:** Bodaboda / Mabasi cargo handler at Magufuli Bus Terminal.

## 3. Core Features

### 3.1 Kariakoo Gold Supplier Verification
- Tier 1 - Basic: Phone + NIDA verified
- Tier 2 - Gold Supplier: BRELA Reg No. API check, TRA TIN check, Physical Shop Verification (field agent + geo-tagged photos + video)
- Badges: "Muagizaji Aliyethibitishwa / Verified Importer", "Mtengenezaji wa Ndani / Local Manufacturer", "Miaka 5+ Kariakoo"
- Trust Score: On-time dispatch %, dispute rate, response time

### 3.2 Bulk Pricing & RFQ
- Tiered MOQ pricing table per SKU. Ex: 1-11 pcs: 25,000 TZS, 12-99: 21,000 TZS, 100+: 18,000 TZS
- RFQ Portal: Buyer posts: Bidhaa, Qty, Budget, Delivery Mkoa. Suppliers bid in 48hr. Auto-match.

### 3.3 Trade Assurance Escrow - "Malipo Salama"
- Escrow wallet held by licensed PSP: ClickPesa / Pesapal
- Buyer pays via M-Pesa / Tigo Pesa / Airtel Money / HaloPesa / Bank (CRDB/NMB)
- Funds held until: Buyer confirms delivery, or auto-release D+7
- Dispute escrow: Photo evidence, 72hr mediation

### 3.4 Logistics Engine
- Tier 1: Ndani ya Dar - Boda/Bajaji, 2-4hr, TZS 5,000-15,000
- Tier 2: Pickup Point - 80+ agent shops in regions
- Tier 3: Mabasi ya Mikoani - Magufuli Terminal integration. Bus name, plate, driver phone, seat/cargo no. SMS tracking to buyer.
- Tier 4: Cargo Trucks - LTL bulk

### 3.5 Negotiation Chat
- Alibaba-style IM with order context
- Inline Price Offer: Supplier sends custom invoice in chat, buyer accepts with 1-tap
- Swahili/English toggle per chat
- Templates: "Bei ya mwisho?", "Nipo tayari kuchukua 200pcs"
